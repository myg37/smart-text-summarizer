# 🤖 Smart Text Summarizer

An AI-powered multilingual text summarization web application, 
developed as my university final project in 2024.
Achieved a perfect score of 100/100.

## 📌 About This Project
This was my final project for my university degree in 2024.
The application allows users to select any text and get an 
AI-powered summary in their preferred language using OpenAI's 
GPT-3.5-turbo model.

## ✨ Features
- AI-powered text summarization with GPT-3.5-turbo
- Multilingual support (ISO 639 language codes)
- User registration and login system
- JWT authentication
- bcrypt password security
- Animated modern UI

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Backend: Node.js, Express.js
- Database: PostgreSQL, NeDB
- AI: OpenAI GPT-3.5-turbo API
- Security: JWT, bcrypt
